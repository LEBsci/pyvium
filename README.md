# PYVIUM CORE

Simple Python wrapper around the "Software development driver DLL" for IviumSoft.

## Installation

Install PYVIUM CORE easily with pip:

```
pip install pyvium-core
```

Or with poetry:

```
poetry add pyvium-core
```
